2. Pre-requisites

  • Basic understanding of HTML, CSS, and JavaScript
  • Understanding of Angular
  • Basic understanding of web standards, DOM etc.

This coding activity starts from scratch involving the set up of the environment and the creation of the web component and the usage of webcomponent in another framework, therefore, there are no starter files required to get started with.
